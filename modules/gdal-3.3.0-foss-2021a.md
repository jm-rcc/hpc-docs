# GDAL 3.3.0  
![Static Badge](https://img.shields.io/badge/class-data-D7D1CC)  
  
  
GDAL is a translator library for raster geospatial data formats that is released under an X/MIT style
 Open Source license by the Open Source Geospatial Foundation. As a library, it presents a single abstract data model
 to the calling application for all supported formats. It also comes with a variety of useful commandline utilities for
 data translation and processing.  
  
  
## Homepage  
The homepage for GDAL 3.3.0 
[https://www.gdal.org](https://www.gdal.org)  
### Usage  
To use the module on Bunya, load it with this command:  
  **`module load gdal/3.3.0-foss-2021a`**
