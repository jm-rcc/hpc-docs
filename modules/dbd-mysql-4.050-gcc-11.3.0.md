# DBD-mysql 4.050  
![Static Badge](https://img.shields.io/badge/class-data-D7D1CC)  
  
  
Perl binding for MySQL  
  
  
### Usage  
To use the module on Bunya, load it with the command  
  **`module load dbd-mysql/4.050-gcc-11.3.0`**.  
  
### Toolchain  
GCC (11.3.0)  
  
### Toolchain Options  
None
