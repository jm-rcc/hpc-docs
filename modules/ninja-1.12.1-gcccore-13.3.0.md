# Ninja 1.12.1  
![Static Badge](https://img.shields.io/badge/class-tools-D7D1CC)  
  
  
Ninja is a small build system with a focus on speed.  
  
  
### Usage  
To use the module on Bunya, load it with the command  
  **`module load ninja/1.12.1-gcccore-13.3.0`**.  
  
### Toolchain
