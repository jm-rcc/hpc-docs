# SPAdes 3.15.5  
![Static Badge](https://img.shields.io/badge/class-bio-D7D1CC)  
  
  
Genome assembler for single-cell and isolates data sets  
  
  
### Usage  
To use the module on Bunya, load it with the command  
  **`module load spades/3.15.5-gcc-11.3.0`**.  
  
### Toolchain  
GCC-11.3.0  
  
### Toolchain Options  
None
