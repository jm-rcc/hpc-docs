# jemalloc 5.3.0  
![Static Badge](https://img.shields.io/badge/class-lib-D7D1CC)  
  
  
jemalloc is a general purpose malloc(3) implementation that emphasizes fragmentation avoidance and
 scalable concurrency support.  
  
  
## Homepage  
The homepage for jemalloc 5.3.0   
[https://jemalloc.net](https://jemalloc.net)  
### Usage  
To use the module on Bunya, load it with this command:  
  **`module load jemalloc/.5.3.0-gcccore-11.3.0`**
