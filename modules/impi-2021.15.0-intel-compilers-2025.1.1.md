# impi 2021.15.0  
![Static Badge](https://img.shields.io/badge/class-mpi-D7D1CC)  
  
  
Intel MPI Library, compatible with MPICH ABI  
  
  
### Usage  
To use the module on Bunya, load it with the command  
  **`module load impi/2021.15.0-intel-compilers-2025.1.1`**.  
  
### Toolchain  
intel-compilers-2025.1.1  
  
### Toolchain Options  
None  
  
  
## Available on architecture types  
- epyc3 a100  
- epyc3 l40  
- epyc4 a16  
- xeonsp4 h100  
  
  
## Dependencies  
[UCX 1.18.0  (GCCcore-14.2.0)](ucx-.1.18.0-gcccore-14.2.0.md)
