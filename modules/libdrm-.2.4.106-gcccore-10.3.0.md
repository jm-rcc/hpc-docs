# libdrm 2.4.106  
![Static Badge](https://img.shields.io/badge/class-lib-D7D1CC)  
  
  
Direct Rendering Manager runtime library.  
  
  
### Usage  
To use the module on Bunya, load it with the command  
  **`module load libdrm/.2.4.106-gcccore-10.3.0`**.  
  
### Toolchain  
GCCcore (10.3.0)  
  
### Toolchain Options
