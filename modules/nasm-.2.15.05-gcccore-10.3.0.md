# NASM 2.15.05  
![Static Badge](https://img.shields.io/badge/class-lang-D7D1CC)  
  
  
NASM: General-purpose x86 assembler  
  
  
### Usage  
To use the module on Bunya, load it with the command  
  **`module load nasm/.2.15.05-gcccore-10.3.0`**.  
  
### Toolchain  
GCCcore-10.3.0  
  
### Toolchain Options  
None
