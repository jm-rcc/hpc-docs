# impi 2021.2.0  
![Static Badge](https://img.shields.io/badge/class-mpi-D7D1CC)  
  
  
Intel MPI Library, compatible with MPICH ABI  
  
  
## Homepage  
The homepage for impi 2021.2.0   
[https://software.intel.com/content/www/us/en/develop/tools/mpi-library.html](https://software.intel.com/content/www/us/en/develop/tools/mpi-library.html)  
### Usage  
To use the module on Bunya, load it with this command:  
  **`module load impi/2021.2.0-intel-compilers-2021.2.0`**  
  
### Toolchain  
intel-compilers-2021.2.0
