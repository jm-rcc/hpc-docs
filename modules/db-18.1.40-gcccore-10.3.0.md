# DB 18.1.40  
![Static Badge](https://img.shields.io/badge/class-tools-D7D1CC)  
  
  
Berkeley DB enables the development of custom data management
 solutions, without the overhead traditionally associated with such custom
 projects.  
  
  
## Homepage  
The homepage for DB 18.1.40 
[https://www.oracle.com/technetwork/products/berkeleydb](https://www.oracle.com/technetwork/products/berkeleydb)  
### Usage  
To use the module on Bunya, load it with this command:  
  **`module load db/18.1.40-gcccore-10.3.0`**
