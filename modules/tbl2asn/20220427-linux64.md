# tbl2asn 20220427  
(bio)  
  
  
Tbl2asn is a command-line program that automates the creation of
 sequence records for submission to GenBank  
  
  
### Toolchain  
None  
  
### Toolchain Options  
None  
  
  
### Usage  
Use the command **`module load tbl2asn/20220427-linux64`** to load the module on Bunya.

- epyc4 a16  
- epyc4 l40s
