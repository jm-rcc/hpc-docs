# SQLite 3.35.4  
![Static Badge](https://img.shields.io/badge/class-devel-D7D1CC)  
  
  
SQLite: SQL Database Engine in a C Library  
  
  
### Usage  
To use the module on Bunya, load it with the command  
  **`module load sqlite/.3.35.4-gcccore-10.3.0`**.  
  
### Toolchain  
GCCcore (10.3.0)  
  
### Toolchain Options  
{'pic': True}
