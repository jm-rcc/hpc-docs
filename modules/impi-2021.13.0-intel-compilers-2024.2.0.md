# impi 2021.13.0  
![Static Badge](https://img.shields.io/badge/class-mpi-D7D1CC)  
  
  
Intel MPI Library, compatible with MPICH ABI  
  
  
### Usage  
To use the module on Bunya, load it with the command  
  **`module load impi/2021.13.0-intel-compilers-2024.2.0`**.  
  
### Toolchain  
intel-compilers-2024.2.0  
  
### Toolchain Options  
None  
  
  
## Available on architecture types  
- epyc3 a100  
- epyc3 h100  
- epyc3 l40  
- epyc3 mi210  
- epyc3  
- epyc4 a16  
- epyc4 l40s  
- epyc4 mi210  
- epyc4  
- xeonsp4 h100  
- xeonsp4
