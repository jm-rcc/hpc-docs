# ScaFaCoS 1.0.4  
![Static Badge](https://img.shields.io/badge/class-math-D7D1CC)  
  
  
ScaFaCoS is a library of scalable fast coulomb solvers.  
  
  
### Usage  
To use the module on Bunya, load it with the command  
  **`module load scafacos/1.0.4-foss-2023a`**.  
  
### Toolchain  
foss-2023a  
  
### Toolchain Options  
{'usempi': True}
