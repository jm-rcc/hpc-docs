# xorg-macros 1.20.0  
![Static Badge](https://img.shields.io/badge/class-devel-D7D1CC)  
  
  
X.org macros utilities.  
  
  
### Usage  
To use the module on Bunya, load it with the command  
  **`module load xorg-macros/.1.20.0-gcccore-12.3.0`**.  
  
### Toolchain
