# STAR 2.7.10b  
(bio)  
  
  
STAR aligns RNA-seq reads to a reference genome using uncompressed suffix arrays.  
  
  
### Toolchain  
{'name': 'GCC', 'version': '11.3.0'}  
  
### Toolchain Options  
{'openmp': True}  
  
  
### Usage  
Use the command **`module load star/2.7.10b-gcc-11.3.0`** to load the module on Bunya.
