# XML-LibXML 2.0207  
![Static Badge](https://img.shields.io/badge/class-data-D7D1CC)  
  
  
Perl binding for libxml2  
  
  
### Usage  
To use the module on Bunya, load it with the command  
  **`module load xml-libxml/2.0207-gcccore-10.3.0`**.  
  
### Toolchain  
GCCcore-10.3.0  
  
### Toolchain Options  
None
