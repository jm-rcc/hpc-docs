# libyaml 0.2.5  
![Static Badge](https://img.shields.io/badge/class-lib-D7D1CC)  
  
  
LibYAML is a YAML parser and emitter written in C.  
  
  
### Usage  
To use the module on Bunya, load it with the command  
  **`module load libyaml/.0.2.5-gcccore-11.3.0`**.  
  
### Toolchain  
GCCcore-11.3.0  
  
### Toolchain Options  
None
