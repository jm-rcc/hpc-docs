# maeparser 1.3.0  
![Static Badge](https://img.shields.io/badge/class-tools-D7D1CC)  
  
  
maeparser is a parser for Schrodinger Maestro files.  
  
  
### Usage  
To use the module on Bunya, load it with the command  
  **`module load maeparser/1.3.0-gompi-2022a`**.  
  
### Toolchain  
gompi (2022a)  
  
### Toolchain Options  
None

- epyc3  
- epyc4 a16
