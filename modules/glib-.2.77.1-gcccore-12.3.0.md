# GLib 2.77.1  
![Static Badge](https://img.shields.io/badge/class-vis-D7D1CC)  
  
  
GLib is one of the base libraries of the GTK+ project  
  
  
### Usage  
To use the module on Bunya, load it with the command  
  **`module load glib/.2.77.1-gcccore-12.3.0`**.  
  
### Toolchain  
GCCcore (12.3.0)  
  
### Toolchain Options
