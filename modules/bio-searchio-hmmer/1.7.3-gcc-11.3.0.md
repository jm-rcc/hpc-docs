# Bio-SearchIO-hmmer 1.7.3  
(bio)  
  
Code to parse output from hmmsearch, hmmscan, phmmer and nhmmer, compatible
with both version 2 and version 3 of the HMMER package from http://hmmer.org.  
----  
### Toolchain  
{'name': 'GCC', 'version': '11.3.0'}  
### Toolchain Options  
None
