# util-linux 2.36  
![Static Badge](https://img.shields.io/badge/class-tools-D7D1CC)  
  
  
Set of Linux utilities  
  
  
### Usage  
To use the module on Bunya, load it with the command  
  **`module load util-linux/.2.36-gcccore-10.3.0`**.  
  
### Toolchain  
GCCcore (10.3.0)  
  
### Toolchain Options
