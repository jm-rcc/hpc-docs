# intltool 0.51.0  
![Static Badge](https://img.shields.io/badge/class-devel-D7D1CC)  
  
  
intltool is a set of tools to centralize translation of
 many different file formats using GNU gettext-compatible PO files.  
  
  
## Homepage  
The homepage for intltool 0.51.0 
[https://freedesktop.org/wiki/Software/intltool/](https://freedesktop.org/wiki/Software/intltool/)  
### Usage  
To use the module on Bunya, load it with this command:  
  **`module load intltool/.0.51.0-gcccore-11.3.0`**
