# lpsolve 5.5.2.11  
![Static Badge](https://img.shields.io/badge/class-math-D7D1CC)  
  
  
Mixed Integer Linear Programming (MILP) solver  
  
  
### Usage  
To use the module on Bunya, load it with the command  
  **`module load lpsolve/5.5.2.11-gcc-10.3.0`**.  
  
### Toolchain  
GCC (10.3.0)  
  
### Toolchain Options  
None
