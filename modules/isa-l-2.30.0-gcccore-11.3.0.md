# ISA-L 2.30.0  
![Static Badge](https://img.shields.io/badge/class-lib-D7D1CC)  
  
  
Intelligent Storage Acceleration Library  
  
  
### Usage  
To use the module on Bunya, load it with the command  
  **`module load isa-l/2.30.0-gcccore-11.3.0`**.  
  
### Toolchain  
GCCcore (11.3.0)  
  
### Toolchain Options  
None

- epyc3  
- epyc4 a16
