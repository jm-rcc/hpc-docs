# HarfBuzz 5.3.1  
![Static Badge](https://img.shields.io/badge/class-vis-D7D1CC)  
  
  
HarfBuzz is an OpenType text shaping engine.  
  
  
### Usage  
To use the module on Bunya, load it with the command  
  **`module load harfbuzz/.5.3.1-gcccore-12.3.0`**.  
  
### Toolchain  
GCCcore (12.3.0)  
  
### Toolchain Options  
None
