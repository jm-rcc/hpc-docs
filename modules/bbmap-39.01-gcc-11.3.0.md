# BBMap 39.01  
![Static Badge](https://img.shields.io/badge/class-bio-D7D1CC)  
  
  
BBMap short read aligner, and other bioinformatic tools.  
  
  
### Usage  
To use the module on Bunya, load it with the command  
  **`module load bbmap/39.01-gcc-11.3.0`**.  
  
### Toolchain  
GCC (11.3.0)  
  
### Toolchain Options  
None
