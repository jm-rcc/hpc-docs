# Clang 12.0.1  
(compiler)  
  
  
C, C++, Objective-C compiler, based on LLVM.  Does not
 include C++ standard library -- use libstdc++ from GCC.  
  
  
### Toolchain  
{'name': 'GCCcore', 'version': '10.3.0'}  
  
### Toolchain Options  
None  
  
  
### Usage  
Use the command **`module load clang/12.0.1-gcccore-10.3.0`** to load the module on Bunya.  
  
## Available on architecture types  
- epyc3  
- epyc4  
- xeonsp4
