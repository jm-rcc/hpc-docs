# R 4.2.1  
![Static Badge](https://img.shields.io/badge/class-lang-D7D1CC)  
  
  
R is a free software environment for statistical computing
 and graphics.  
  
  
### Usage  
To use the module on Bunya, load it with the command  
  **`module load r/4.2.1-foss-2022a`**.
