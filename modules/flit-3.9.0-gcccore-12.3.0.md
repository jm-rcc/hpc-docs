# flit 3.9.0  
![Static Badge](https://img.shields.io/badge/class-tools-D7D1CC)  
  
  
A simple packaging tool for simple packages.  
  
  
### Usage  
To use the module on Bunya, load it with the command  
  **`module load flit/3.9.0-gcccore-12.3.0`**.  
  
### Toolchain  
GCCcore (12.3.0)  
  
### Toolchain Options  
{'pic': True}
