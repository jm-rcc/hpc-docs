# gnuplot 5.4.4  
![Static Badge](https://img.shields.io/badge/class-vis-D7D1CC)  
  
  
Portable interactive, function plotting utility  
  
  
### Usage  
To use the module on Bunya, load it with the command  
  **`module load gnuplot/.5.4.4-gcccore-11.3.0`**.  
  
### Toolchain  
GCCcore (11.3.0)  
  
### Toolchain Options  
None
