# fastp 0.23.2  
(bio)  
  
A tool designed to provide fast all-in-one preprocessing for FastQ files.
 This tool is developed in C++ with multithreading supported to afford high performance.  
----  
### Toolchain  
{'name': 'GCC', 'version': '11.3.0'}
