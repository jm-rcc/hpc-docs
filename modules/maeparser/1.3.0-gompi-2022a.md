# maeparser 1.3.0  
(tools)  
  
  
maeparser is a parser for Schrodinger Maestro files.  
  
  
### Toolchain  
{'name': 'gompi', 'version': '2022a'}

Use the command **`module load maeparser/1.3.0-gompi-2022a`** to load the module on Bunya.
