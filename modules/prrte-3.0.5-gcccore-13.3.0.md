# PRRTE 3.0.5  
![Static Badge](https://img.shields.io/badge/class-lib-D7D1CC)  
  
  
PRRTE is the PMIx Reference RunTime Environment  
  
  
### Usage  
To use the module on Bunya, load it with the command  
  **`module load prrte/3.0.5-gcccore-13.3.0`**.  
  
### Toolchain  
GCCcore (13.3.0)  
  
### Toolchain Options  
{'pic': True}
