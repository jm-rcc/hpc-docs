# MrBayes 3.2.7a 
![Static Badge](https://img.shields.io/badge/class-bio-D7D1CC)  
  
  
MrBayes is a program for the Bayesian estimation of phylogeny.  
  
  
## Homepage

[http://mrbayes.csit.fsu.edu](http://mrbayes.csit.fsu.edu)
