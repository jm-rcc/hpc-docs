# ffnvcodec 12.0.16.0  
(lib)  
  
  
FFmpeg nvidia headers. Adds support for nvenc and nvdec. Requires Nvidia GPU and drivers to be present
(picked up dynamically).  
  
  
### Toolchain  
None  
  
### Toolchain Options  
None  
  
  
Use the command **`module load ffnvcodec/12.0.16.0`** to load the module on Bunya.
