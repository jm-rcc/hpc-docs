# intel 2021a  
(toolchain)  
  
  
Compiler toolchain including Intel compilers, Intel MPI and Intel Math Kernel Library (MKL).  
  
  
### Toolchain  
None  
  
### Toolchain Options  
None  
  
  
Use the command **`module load intel/2021a`** to load the module on Bunya.  
  
## Available on architecture types  
- epyc3 a100  
- epyc3 h100  
- epyc3 l40  
- epyc3 mi210  
- epyc3  
- epyc4 a16  
- epyc4 l40s  
- epyc4 mi210  
- epyc4  
- xeonsp4 h100  
- xeonsp4
