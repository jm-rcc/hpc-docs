# intel 2025a  
(toolchain)  
  
  
Compiler toolchain including Intel compilers, Intel MPI and Intel Math Kernel Library (MKL).  
  
  
### Toolchain  
None  
  
### Toolchain Options  
None  
  
  
Use the command **`module load intel/2025a`** to load the module on Bunya.  
  
## Available on architecture types  
- epyc3 a100  
- epyc3 l40  
- epyc4 a16  
- xeonsp4 h100
