# libarchive 3.6.2  
![Static Badge](https://img.shields.io/badge/class-tools-D7D1CC)  
  
  

 Multi-format archive and compression library
  
  
  
### Usage  
To use the module on Bunya, load it with the command  
  **`module load libarchive/3.6.2-gcccore-12.3.0`**.  
  
### Toolchain  
GCCcore (12.3.0)  
  
### Toolchain Options  
None

- epyc3
