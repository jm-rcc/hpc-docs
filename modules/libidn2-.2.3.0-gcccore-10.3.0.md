# libidn2 2.3.0 
![Static Badge](https://img.shields.io/badge/class-lib-D7D1CC)  
  
  
Libidn2 implements the revised algorithm for internationalized domain names called IDNA2008/TR46.  
  
  
## Homepage  
The homepage for libidn2 2.3.0  
[http://www.gnu.org/software/libidn2](http://www.gnu.org/software/libidn2)  
### Usage  
To use this module on Bunya, load it with this command:  
  **`module load libidn2/.2.3.0-gcccore-10.3.0`**
