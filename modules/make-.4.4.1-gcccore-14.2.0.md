# make 4.4.1  
![Static Badge](https://img.shields.io/badge/class-devel-D7D1CC)  
  
  
GNU version of make utility  
  
  
### Usage  
To use the module on Bunya, load it with the command  
  **`module load make/.4.4.1-gcccore-14.2.0`**.  
  
### Toolchain  
GCCcore (14.2.0)  
  
### Toolchain Options  
None

- epyc3 l40  
- epyc4 a16  
- xeonsp4 h100
