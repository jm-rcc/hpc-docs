# libpng 1.6.37  
![Static Badge](https://img.shields.io/badge/class-lib-D7D1CC)  
  
  
libpng is the official PNG reference library  
  
  
### Usage  
To use the module on Bunya, load it with the command  
  **`module load libpng/.1.6.37-gcccore-11.3.0`**.  
  
### Toolchain  
GCCcore-11.3.0  
  
### Toolchain Options  
{'pic': True}
