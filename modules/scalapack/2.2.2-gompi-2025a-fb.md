# ScaLAPACK 2.2.2  
(numlib)  
  
  
The ScaLAPACK (or Scalable LAPACK) library includes a subset of LAPACK routines
 redesigned for distributed memory MIMD parallel computers.  
  
  
### Toolchain  
{'name': 'gompi', 'version': '2025a'}  
  
### Toolchain Options  
{'extra_fflags': '-lpthread', 'openmp': True, 'pic': True, 'usempi': True}  
  
  
Use the command **`module load scalapack/2.2.2-gompi-2025a-fb`** to load the module on Bunya.  
  
## Available on architecture types  
- epyc3 a100  
- epyc3 l40  
- epyc4 a16  
- xeonsp4 h100  
  
  
## Dependencies  
[('CMake', '3.31.3')]
