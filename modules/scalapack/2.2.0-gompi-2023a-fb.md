# ScaLAPACK 2.2.0  
(numlib)  
  
The ScaLAPACK (or Scalable LAPACK) library includes a subset of LAPACK routines
 redesigned for distributed memory MIMD parallel computers.  
----  
### Toolchain  
{'name': 'gompi', 'version': '2023a'}  
### Toolchain Options  
{'extra_fflags': '-lpthread', 'openmp': True, 'pic': True, 'usempi': True}  
----  
Use the command **`module load scalapack/2.2.0-gompi-2023a-fb`** to load the module on Bunya.  
## Available on architecture types  
- epyc3 a100  
- epyc3 h100  
- epyc3 l40  
- epyc3 mi210  
- epyc3  
- epyc4 a16  
- epyc4 l40s  
- epyc4 mi210
