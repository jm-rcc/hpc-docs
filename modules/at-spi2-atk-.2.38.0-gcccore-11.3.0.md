# at-spi2-atk 2.38.0  
![Static Badge](https://img.shields.io/badge/class-vis-D7D1CC)  
  
  
AT-SPI 2 toolkit bridge  
  
  
### Usage  
To use the module on Bunya, load it with the command  
  **`module load at-spi2-atk/.2.38.0-gcccore-11.3.0`**.  
  
### Toolchain  
GCCcore (11.3.0)  
  
### Toolchain Options  
None
