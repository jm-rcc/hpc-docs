# SpectrA 1.0.1  
![Static Badge](https://img.shields.io/badge/class-math-D7D1CC)  
  
  
Spectra stands for Sparse Eigenvalue Computation Toolkit as a Redesigned ARPACK. It is a C++
 library for large scale eigenvalue problems, built on top of Eigen, an open source linear algebra library.  
  
  
## Homepage  
The homepage for SpectrA 1.0.1 
[https://spectralib.org/](https://spectralib.org/)  
### Usage  
To use the module on Bunya, load it with this command:  
  **`module load spectra/1.0.1-gcccore-12.3.0`**  
  
### Toolchain  
GCCcore-12.3.0
