# libpsl 0.21.1  
![Static Badge](https://img.shields.io/badge/class-lib-D7D1CC)  
  
  
C library for the Public Suffix List  
  
  
### Usage  
To use the module on Bunya, load it with the command  
  **`module load libpsl/.0.21.1-gcccore-11.3.0`**.  
  
### Toolchain  
GCCcore-11.3.0
