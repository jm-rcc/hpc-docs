# GMAP-GSNAP 2023-02-17

GMAP: A Genomic Mapping and Alignment Program for mRNA and EST Sequences

### Usage  
To use the module on Bunya, load it with the command  
  **`module load gmap-gsnap/2023-02-17-gcc-11.3.0`**.
