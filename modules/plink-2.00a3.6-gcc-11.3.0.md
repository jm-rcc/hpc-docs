# PLINK 2.00a3.6  
![Static Badge](https://img.shields.io/badge/class-bio-D7D1CC)  
  
  
Whole-genome association analysis toolset  
  
  
### Usage  
To use the module on Bunya, load it with the command  
  **`module load plink/2.00a3.6-gcc-11.3.0`**.  
  
### Toolchain  
GCC (11.3.0)  
  
### Toolchain Options  
None
