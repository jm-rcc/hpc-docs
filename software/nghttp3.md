# nghttp3  
## Installed versions  
 - nghttp3/.0.6.0-gcccore-11.3.0  
  
## Description  
nghttp3 is an implementation of RFC 9114 HTTP/3
mapping over QUIC and RFC 9204 QPACK in C.
It does not depend on any particular QUIC transport implementation.
