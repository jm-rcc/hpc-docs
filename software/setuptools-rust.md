# setuptools-rust  
## Installed versions  
 - setuptools-rust/1.6.0-gcccore-12.3.0  (version), (toolchain)  
### Description  
setuptools-rust is a plugin for setuptools to build Rust Python extensions
implemented with PyO3 or rust-cpython.
