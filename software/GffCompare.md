# GffCompare  
## Installed versions  
 - gffcompare/0.12.2-gcc-10.3.0  
  
## Description  
GffCompare provides classification and reference annotation mapping and
 matching statistics for RNA-Seq assemblies (transfrags) or other generic GFF/GTF files.
