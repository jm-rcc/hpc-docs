# CFITSIO  
## Installed versions  
 - cfitsio/4.3.0-gcccore-12.3.0  (version), (toolchain)  
  
## Description  
CFITSIO is a library of C and Fortran subroutines for reading and writing data files in
FITS (Flexible Image Transport System) data format.
