# gc  
## Installed versions  
 - gc/.8.2.2-gcccore-11.3.0  (version), (toolchain)  
  
## Description  
The Boehm-Demers-Weiser conservative garbage collector can be used as a
garbage collecting replacement for C malloc or C++ new.
