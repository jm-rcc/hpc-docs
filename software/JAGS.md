# JAGS  
## Installed versions  
 - jags/4.3.0-foss-2021a  (version), (toolchain)  
  
## Description  
JAGS is Just Another Gibbs Sampler.  It is a program for analysis
 of Bayesian hierarchical models using Markov Chain Monte Carlo (MCMC) simulation
